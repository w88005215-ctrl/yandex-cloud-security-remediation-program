# Publication Artifact Curation

## Purpose

This document records the final public-repository curation pass for the cloud security remediation portfolio project.

The curation pass removed low-value intermediate execution artifacts from the current public tree while preserving the final portfolio evidence set.

## Retained public evidence classes

- Final English and Russian reports.
- Final remediation metrics.
- Final control matrix.
- Final risk register.
- Final cross-cloud control mapping.
- Final repository audit and sanitization reports.
- Final project completion reports.
- Successful OIDC, Checkov, Gitleaks, Kyverno, SBOM, vulnerability, cleanup, and audit evidence.
- Redaction notes and IaC security exception rationale.

## Removed artifact class

The removed files were process-noise artifacts: repeated git-control snapshots, terminal recovery remnants, append-closeout traces, and intermediate repair logs.

The exact removed-file manifest is intentionally not retained in the public repository because keeping that manifest would preserve references to the noise artifacts. A private copy was preserved outside the repository before cleanup.

## Result

- Removed public noise artifact count: 60
- Public stale-reference cleanup: completed
- Cloud actions: none
- Git history rewrite: none
- Force push: none

# Курирование публикационных артефактов

## Назначение

Этот документ фиксирует финальную очистку публичной версии проекта от низкоценного технического шума.

В публичной версии сохранены финальные отчеты, метрики, матрицы, risk register, cross-cloud mapping, evidence по успешным проверкам, отчеты по санитизации, redaction notes и обоснования IaC security exceptions.

## Что удалено

Удалены промежуточные процессные артефакты: повторяющиеся git-control snapshots, следы терминального восстановления, append-closeout traces и промежуточные repair-логи.

Точный список удаленных файлов не публикуется, чтобы не сохранять в публичной версии ссылки на шумовые артефакты. Приватная копия сохранена вне репозитория.

## Результат

- Удалено шумовых публичных артефактов: 60
- Очистка устаревших ссылок: выполнена
- Cloud actions: none
- Git history rewrite: none
- Force push: none
